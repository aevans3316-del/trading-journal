Sync & Backup
Keep your phone and laptop on the same journal with GitHub sync
☁️ GitHub Sync
Your trades and screenshots are saved to a private GitHub repository, so every device that connects with your token sees the same journal. Create a token at github.com → Settings → Developer settings → Personal access tokens (classic) with the repo scope, then paste it here. Do this once on each device.
If this repo doesn't exist yet it will be created as private automatically.
Syncs automatically when you open the app and whenever you save or delete a trade. Your token is stored only in this browser — treat it like a password.
💾 Export
Offline backup: downloads a single file with every trade and screenshot.
📥 Import
Load a backup file exported from your other device. Adds any trades you don't already have.
🗑 Danger Zone
Permanently delete every trade and screenshot on this device.
